## To run

`dotnet run`

or

`dotnet watch run`

## To create migrations

- EF Core Migrations: `dotnet ef migrations add InitialMigration`

## To create tables

`dotnet ef database update`

## Routes

Accessing `http://localhost:5064/swagger/index.html`
